# Dictionary of greetings in five languages

A named list containing common greetings in English, French, Spanish,
German and Italian. Each element of the list is a character vector with
greetings in that language.

## Usage

``` r
dictionary
```

## Format

A named list with 5 elements:

- English:

  Character vector of English greetings.

- French:

  Character vector of French greetings.

- Spanish:

  Character vector of Spanish greetings.

- German:

  Character vector of German greetings.

- Italian:

  Character vector of Italian greetings.
